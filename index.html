ℹ️ About RouteMaster

        
      

      


        

          
🏭 What is RouteMaster?

          
RouteMaster is an AI-powered warehouse navigation simulator that calculates optimal robot
            routes through warehouse environments using industry-standard pathfinding algorithms.

        


        

          
✨ Features

          

            
📊 Interactive warehouse grid

            
🧮 BFS & A* algorithms

            
🤖 Real-time route animation

            
🖱️ Drag-and-drop cells

            
📁 CSV data import

            
📸 Grid export (PNG/Word)

            
📜 Route history tracking

            
📡 AI shelf scanning

            
📈 Efficiency metrics

            
🏗️ 3D warehouse view

            
🚶 Street-view mode

            
🎲 Random grid generator

          

        


        

          
📋 How to Use

          
    
            
  1. Configure Grid — Enter JSON, upload CSV, or click Generate Random
    2. 
            
  2. Set Positions — Drag the blue robot (start) and orange crate (target)
    3. 
            
  3. Add Obstacles — Click empty cells to toggle shelves on/off
    4. 
            
  4. Choose Algorithm — Toggle between BFS and A* using the switch
    5. 
            
  5. Compute Route — Click "Compute Optimal Route" to find the path
    6. 
            
  6. Watch Animation — See the robot navigate through the warehouse
    7. 
            
  7. Review Metrics — Check steps, distance, time, and efficiency
    8. 
            
  8. Export & Save — Download as PNG, Word, or save to history
    9. 
          

        


        

          
💡 Quick Tips

          

            
🔍 Use Scan Warehouse to auto-detect the target and trigger routing

            
🏗️ Switch to 3D View to see your warehouse from above with orbit controls
            

            
🚶 Use Street View after computing a route for first-person walkthrough
            

            
📜 Open Route History to reload past grids or download their snapshots
